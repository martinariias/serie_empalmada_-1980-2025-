**Serie histórica del PBI de Argentina (1980–2025)**

Este proyecto tiene como objetivo construir una serie trimestral larga del PBI de Argentina a precios constantes mediante el empalme de dos series estadísticas con distinta base metodológica, se exportaron las series limpias con el fin de comprobar los datos

**Metodología**

* Se utilizaron dos series de PBI a precios constantes y se limpiaron
* Se empalmaron mediante el método de coeficiente de enlace, exportando las series empalmadas a un excel.
* Se construyó una serie homogénea 1980–2025.
* La serie fue desestacionalizada.
* Se analizó la tendencia de largo plazo y los ciclos económicos.
* Se graficó la evolución del PBI trimestral.

**Herramientas utilizadas**
* Python
  Pandas
  Matplotlib
* Excel
* Estadistica

**Objetivos**

* Limpieza de datos
* Analizar crecimiento económico
* Analizar ciclos económicos
*Modelos econométricos
* Forecast del PBI
* Análisis macroeconómico
